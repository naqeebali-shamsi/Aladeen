# FAQ

## Is this a plugin or extension?

No. It's a process — a documented SOP with prompts and templates. You paste the prompts into Claude Code and follow the procedure. Nothing to install.

## Do I need to be a Claude Code expert?

You should have a working Claude Code setup that's gotten messy. If you've installed MCP servers, hooks, skill packs, or custom memory systems and things feel bloated or confusing, this is for you. If you just installed Claude Code yesterday, you probably don't need this yet.

## How long does the first cleanup take?

30–60 minutes for the full SOP. Monthly maintenance checks take about 5 minutes.

## What results should I expect?

After the cleanup you should see: faster startup (less context loaded), fewer tool conflicts, cleaner separation between global and project config, one owner per capability instead of overlapping tools fighting for the same job, and a setup you can actually explain by reading a few files.

## Will this automatically clean up my setup?

No. It gives you a controlled, step-by-step workflow to audit what you have, decide what stays and what goes, apply changes safely, and verify nothing broke. You stay in control the whole time.

## What if my setup is very custom?

That's exactly what the audit templates and prompts are designed for. The SOP doesn't assume a specific setup — it walks you through inventorying whatever you have and making intentional decisions about each piece.

## Do I get prompts I can copy-paste?

Yes. Three ready-to-use prompts: one for a dry-run audit, one to apply changes, and one for reviewing tools you want to add back later.

## Can I reuse this across multiple machines?

Yes. The SOP is machine-independent. Run it on each machine where you have Claude Code set up. The templates will have different content per machine, but the process is the same.

## What format are the files?

Plain Markdown. Open them in any text editor, VS Code, Obsidian, Notion, or wherever you work. Paste the prompts directly into Claude Code.

## Can I adapt this for my team?

Yes. The templates and SOP are designed to be modified. Add columns, rename sections, build on it.
